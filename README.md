### How To Install & Run

Inside `backend` folder,

- `npm install`
- clone `.env.example` and rename to `.env`
- `npm run start:dev`
- open [http://localhost:5000/doc/](http://localhost:5000/doc/)

Inside `frontend` folder,

- `npm install`
- clone `.env.example` and rename to `.env`
- `npm run start`
- open [http://localhost:3000/](http://localhost:3000/)
